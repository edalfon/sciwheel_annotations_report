Don't know what that is?, or how to get it?

It is basically the way you authorize us to access data from your Sciwheel account. This extension uses Sciwheel's API to download the annotations from the reference of your choice. But "API can only be used to retrieve data from user accounts for which you have a valid access token."

Now, how do you get yours?

Quoting Sciwheel's documentation again:
An access token can be generated by any user by performing the following steps:
  • Sign in to Sciwheel with the relevant user
  • Go to My Account > External API
  • Here you can view/copy your token or generate a new one (please note that generating a new token will invalidate any existing token for this user account)
 Don't worry. You only have to do this once. If you synchronize Chrome with your Google account, we will store the token there and it will be available for you to use this extension without further configuration in other machines. Please note that, while stored locally, the token is not encrypted.
